# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-02-18

### Added

- Initial release
- Authentication commands (login, logout, whoami)
- Package management (publish, install, search, info)
- Access control (public, restricted, grant, revoke, list)
- Organization management (create, list, show, delete, add, remove, packages)
- CI/CD token management (create, list, revoke)
- Setup instructions command
- Configuration management
